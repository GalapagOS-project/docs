## Candidates
- Firefox
- Falkon

## Findings
### Firefox
Pros:
- Better compatability
- Smoother rendering
- Lower memory during YouTube playback
- Better overall responsiveness

Cons:
- Higher memory during light browsing

### Falkon
Pros:
- Lower memory during some browsing workloads

Cons:
- Less responsive
- Worse compatibility, possible issues with Cloudflare and DRM
- Higher memory during video playback
